# ChatGPT Enhancement Suite

A powerful Chrome Extension that supercharges your ChatGPT experience with Auto-rename, Hierarchical Folders, a Prompt Toolbox, and a Quick Learn Window.

> **Note**: This project is based on the open-source [ChatGPT Conversation Timeline](https://github.com/example/repo) extension. We have stripped it down to focus on these core productivity features and added significant new functionality.

## Features

### 1. Auto-rename (Smart Titles)
Automatically generates concise and relevant titles for your chats.

- **Why**: Keeps your history organized without manual renaming.
- **Privacy**: Summarization happens in a separate background tab to avoid context pollution.

### 2. Hierarchical Folders
Organize your chats into a nested folder structure.
- **Drag & Drop**: Easily move chats into folders.
- **Manage**: Create, rename, and delete folders from the sidebar.

### 3. Prompt Toolbox
Access your favorite prompts instantly.
- **Quick Insert**: Click the ‚ö° icon near the input box to browse and insert prompts.
- **Import/Export**: Share your prompt library via JSON.

### 4. Quick Learn Window
Get explanations without losing your place.
- **Contextual**: Select text and ask for an explanation.
- **Non-intrusive**: The answer appears in a floating window, and your scroll position is strictly maintained.

## Installation

1. Clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the `extension` folder.
